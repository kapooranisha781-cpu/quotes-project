Quotes Generator

A simple and interactive Quotes Generator web 
application that displays random inspirational quotes 
with a button click. This project is built using JavaScript and 
focuses on DOM manipulation and event handling.

Features :- 
Generate random quotes instantly
Clean and simple user interface
Easy-to-use button interaction
Lightweight and fast

Technologies Used :-
HTML
CSS
JavaScript (DOM Manipulation)

How It Works :-
A collection of quotes is stored in an array
When the user clicks the button, a random quote is selected
The selected quote is displayed on the screen dynamically

How to Run :-
Clone the repository
Open the project folder
Run the index.html file in your browser
📸 Preview

A simple interface with a button that shows a new quote each time you click it.
